Sing Version 1.0a

tbl manager

This utility program allows tbl files to be modified. The program presents the list of tbl files for selection, inspection and modification.

    • Directory
      This edit field allows the source directory path name to be specified. This directory will be used for all tbl file operations and will be prefixed to all tbl filenames.

    • FileList
      This table presents the list of tbl files currently present in the source directory. A tbl file may be opened by a simple click on its name in this list. Control will be transered directly to the tbl header and data page.

    • ListBar
      This scroll bar allows the tbl file list to be scrolled.

    • FileName
      This edit field defines the filename of the current tbl file. It is set by the click in the file list window. It may be modified by hand for the loading of files in conjunction with the Load push button or when saving new files in conjunction with the Save push button. In all cases it will be prefixed with the source directory.

    • Status
      This edit field displays the status of the load and file operations.

    • Load
      This push button may be activated to load a tbl file by hand. The filename will be taken from the corresponding edit field and prefixed by the source directory value. The various tables will be loaded into the temporary matrixs for inspection and modification.

    • Save
      This push button may be activated to save the contents of the temporary matrixs to a tbl file. The filename will be taken from the corresponding edit field and prefixed by the source directory value.

    • iTable
      This table control displays for inspection and modification the hexadecimal codes of the initialisation sequences. The values must be carefully modified by hand with valid hexadecimal data.

    • aTable
      This table control displays for inspection and modification the hexadecimal codes of the atribut sequences. For further information concerning the meaning of these sequences please consule the accompanying tbl sequences document. These values must be carefully modified by hand with valid hexadecimal data.

    • aBar
      This scroll bar allows the Atribut Table data entries to be scrolled.

    • TblVersion
      This edit field displays the file version number either MP12 or MP14

    • LgTabAtb
      This edit field displays the length of a atribut table entry.

    • LgBufConv
      This edit field displays the conversion buffer size.

    • OfTabInit
      This edit field displays the file offset of the initialisation table comprising three entries.

    • LgTabInit
      This edit field displays the length of a initialisation table entry.

    • OfTabAtb
      This edit field displays the file offset of the atribut table comprising thirty two (32) entries.

    • OfTabConv
      This edit field displays the file offset of the character conversion table comprising 256 bytes.

    • HexTable
      This edit field displays, for inspection and modification, the hexadecimal values of the various printer character conversions. This is a simple patch mechanism where the standard cursor keys allow the cursor item to be moved. Hexadecimal data may be directly input at the cursor position. The mouse may be used to position to any character value.

    • CharTable
      This edit field works in conjunction with the previous control and displays the character values of the hexadecimal codes. The mouse allows positioning to specifiec characters though direct character input is not possible.
