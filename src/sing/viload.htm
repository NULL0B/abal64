accept imageload

This visual form will be presented to allow inspection and modification of the name of the image file to be loaded.

FileName
This edit field allows the name of the image file to be specified.

The configuration variable IMAGEPATH will be used to localise the image if no explicite path information is provided with the filename.

Accept
By pressing the ACCEPT button the indicated image file will be loaded.

Select
This push button gives access to the File Selection mechanisms allowing prompted selection of the image file to be loaded.

Cancel
By pressing the CANCEL button the image load operation will be abandoned.
