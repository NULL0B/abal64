accept image dimensions

This visual form presents the current image dimensions for modifica tion.

Width
The Width edit field allows the new image width in pixels to be specified.
Height
The Height edit field allows the image height in pixels to be speci fied.
Accept
The Accept button allows the image dimension parameters to be appli ed to the current image which will be transformed as required by th e make fit check box value.
Cancel
If the Cancel button is activated then the Image redimensioning operation will be abandoned.
MakeFit
If the MakeFit check box is checked then the image data will be resized, either stretched or squashed, to fit the new dimensions .

If it is not checked then the image will be clipped or backgr ound colour filled where appropriate.
