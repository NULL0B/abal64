This visual form is presented as a result of selecting the EDIT PROPERTIES option of the menu system whilst the MENU EDITOR is a ctive.It allows the menu level parameters to be modified or inspect ed.

LoadName
The FILENAME edit field presents the full path and name of the file to which the menu is to be saved and from which it was loaded.
Identifier
The IDENTIFIER edit field allows inspection and modification of the menu identification within the generated source. In the case of object language generation this will be the class name. For n on object languages this will be the method or function name.
AtX
The POSITION X and Y edit fields comprise the visual coordinate at which the menu is displayed.
Accept
By pressing the ACCEPT button, modifications performed to the values of this form will be affected to the menu item.
Cancel
IF the CANCEL button is pressed any modifications performed will be discarded.
Method
The METHOD edit field allows a generic method to be specifed as the target method to be activated when menu options are selected.
Popup
The POPUP check box controls whether the menu is a permenant menu o r if it will appear in a pop-up fashion.
Hint
The HINT edit field allows a generic menu level hint message to be specified. This HINT will be displayed in the status bar if it is a ctive when no option specific hint is available.
Font
The FONT edit field allows the font in which the menu is to be draw n to be specified.
Style
The STYLE edit field allows a specific menu style to be selected. This is not yet available.
