accept production properties

This visual form will be presented to allow inspection and modifica tion of application model production parameters prior to the activa tion of the corresponding production operation.

UseAed
By pressing on the EDIT button the document source will be presente d for inspection via the Standard Text Editor mechanisms.
View
By pressing on the VIEW button the system navigator will be activat ed allowing the finalised version of the production document to be consulted. Under Twin Server NV will be used. Under UseIT NET SCAPE will be activated.
ModelName
The MODELNAME edit field presents the name of the application model for inspection.
TargetFile
The PRODUCTION filename edit field presents the production target f ilename for modification or inspection.
NoConfirm
The AUTOMATIC check box when checked alleviates the need to validat e the various stages involved in the production of the target and its documentation. When unchecked each stage will present the corresponding visual frame for validation or cancellation.
Build
The FORCE BUILD check box when checked performs production of all partial products in a systematic manner. When unchecked only the entities which are out of date will be produced.
Accept
By pressing the ACCEPT button the production operation defined by t he associated switchs and values will be activated.
Cancel
By clicking on the CANCEL button the production operation will be abandoned.
Verbose
The VERBOSE check box controls the visualisation of the various application components as they are presented for effective produ ction.
TargetType
When the PROGRAM checkbox is checked the production target will be a complete program making use of its sub-component entities.
IsDocument
The DOCUMENT check box when checked allows the production of an HTM L document containing all Application model Images and documentary text from the sub-components. This will be generated to the Docume nt Filename target if one has been specified.
AdFn
The DOCUMENT filename edit field allows the full path and filename of the file into which the production document is to be generated w hen this has been selected by the DOCUMENT checkbox.
DocumentFile
The PRODUCTION FILENAME edit field allows the full path and filenam e of the production target into which the Program or Module produc tion stream will be generated.
Makefile
When this control has been checked the production mechanism will generate a description of the entire project to a standard makefile.
MakefileName
This edit field allows the full path and filename of the file to which the project make file will be generated if the corresponding check control has been activated.
