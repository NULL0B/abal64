Project Editor

The project modelling tool of the ABAL 3 development environment comprises th following subsystems :
  1. a three dimensional workbench on which the project components are presented
  2. a tool bar providing model navigational controls.
  3. a palette bar allowing choice of colours
  4. a status bar for the display of informational messages
  5. and a popup menu system allowing access to the modelling functions.

Each of the subsytems and their component controls will be described in detail in the remainder of this document.

ToolBarControl
The display of the various model editor tools bars is controlled by this button. When the arrow points upwards a click will cause the display of the tool bar to be inhibited. The arrow button will then point downwards in which case a click will cause the tool bar to be redisplayed.

ScrollBar
The project model scroll bar allows the project model camera position to be easily moved in the direction currently selected by the XYZ selection control. By clicking on either of the two scroll bar buttons single step movement in the corresponding direction is possible. A more powerful effect may be applied by dragging the scroll bar up or down.

Fine movement of the project model view is also possible by mouse drag operation within the project model display area. By pressing the left button and moving the mouse the project model will be moved in the corresponding X and Y directions. By pressing the right button and moving the mouse up or down the project model view will be moved along the Z axis.

The project model view along the Z axis towards 0 is considered to be forwards. By pressing the 'B' key on the keyboard the project model view will be backwards along the Z axis towards infinity ( 2 to the power 32 ). Bye pressing the 'F' key on the keyboard the project model will be restored to the forward direction.

The 'U', for Up and 'D' for down, keys allow the project model view to be oriented along the Y axis in a posative or negative direction.

The 'L', for Left and 'R' for right keys allow the project model view to be oriented along the X axis in a posative or negative direction.

Dimension
This selection control allows selection of the axis or direction along which 3D camera scroll bar will operate. When the X direction has been selected the scroll bar will move the project model camera left and right in a horizontal manner. When the Y direction has been selected the project model view may be deplaced along the vertical axis. When the Z direction has been selected the project model camera will move along the Z axis allowing zoom in and zoom out.

PaletteControl
The display of the colour palette bar to the right of the screen is controlled by this button. When the arrow points upwards a click will cause the display of the palette bar to be inhibited. The arrow button will then point downwards in which case a click will cause the palette bar to be redisplayed.

Palette
The Palette bar displays all colours of the currently loaded palette and allows colour selection by a simple click on the required colour. A click with the left button will set the current foreground colour whilst a click with the right button will set the current background colour. Selection of a colour for the project model background is possible in this way, The effect will be visible at one unless a full screen background image has already been affected to the project modeler background.

File
Selection of this menu options gives access to the generic file operations menu whose options are described here below.

New
Selection of this menu option causes the contents of the current model to be reset to empty. All components will be released.

Save
This menu options performs a quick save of the current model to disk without user prompt.

SaveAs
Activation of this menu option gives access to the corresponding model save parameter input dialog box prior to the eventual model save operation.

Open
This menu operation gives access to the corresponding model load parameter input dialog box allowing new project models to opened and added to the list of open project models.

Production
Selection of this menu option gives access to the project production parameter input dialog box allowing control of the project production process.

Drop
Selection of this menu option causes the current project model to be released from the list of open project models.

Quit
This menu option should be selected if you wish to leave the ABAL 3 development tool. If unsaved modifications are detected amongst any of the open project models you will be prompted via the exit check dialog box for the appropriate action required to be taken prior to return to the host operating system or calling process.

View
Selection of this menu option of the main menu gives access to the view or tool operations menu whose options are described below.

Run
Select this option in order to gain access to the debug and runtime control dialog box allowing abal applications to be run, tested and debugged from within the development tool.

Text
Selection of this menu option gives explicite access to the standard text editor for the current model component.

Image
Activation of this menu option gives explicite access to the bitmap editor. allowing inspection and modification of the model image.

Font
Select this menu option to gain access to the font editor.

Properties
Selection of the properties menu option gives access to the 3D component properties dialog box allowing the various properties to be inspected and eventually modified.

Visual
Selection of this menu option gives access to the component specific visual model editor. For forms and document models this will be the visual forms editor. For Image components this will be the bitmap editor and for Source components this will be the standard text editor.

TreeView
Selection of the treeview menu option toggles the treeview mechanism between visible and hidden states.

Application
Selection of this menu option returns control to this project model editor from the vrious other model editor views.

Event
Selection of this menu option gives access to the menu of widget component methods and documentation.

ItemDocument
Selection of this menu option gives access to the Html Editor allowing inspection modification of the documentation associated with the current active widget of the current forms model.

Documentation and Information provided in this way will be collected together along with the information provided at the forms compoent level, to build the forms documentation during the project model production operation.

FormDocument
Selection of this menu option gives access to the Html Editor allowing inspection and modification of the form level documentation for the current form or document component of the current project model.

Options
Selection of this menu options gives access to the general operations menu the options of which are described here below.

Parameters
Selection of this menu option gives access to the General Parameters dialog box allowing the various editor parameters to be inspected and modified. These parameters will be used as default values for all subsequent new widget and other model components as they are created.

Refresh
Selection of this menu option simply causes the development screen to be redraw for any particular reason.

Configure
Activation of this menu option gices access to the Configuration Parameters of the ABAL 3 development tool. These configuration values are read from the configuration file, normally sing.cfg, during the start up and construction phase of the development environment.

ScreenCapture
This menu option may be activated in order to perform a screen capture operation. The entire screen will be captured to a virtual bitmap then the image save dialog box will be presented allowing the target image type and file name to be specified.

Animate
Selection of this menu operation gives access to the Animation Record and Replay control dialog box. Animation recording allows user input events, equally keyboard and mouse events, to the development environment to be recorded to an animation script file. Animation replay allows a recorded script to be re-enacted performing the identical sequence of input events. These features may be used to create input macros or for tutorial purposes.

Project
Selection of the project menu options gives access to the following project model specific menu functions

Camera
Selction of this menu option gives access to the Camera Properties dialog box allowing inspection and modification of the project model 3D camera operational parameters.

Translate
Selection of this menu operation will launch a national language translation operation on a global scale. All national language message text payloads will be injected into the translation dictionary mechanisms and then reaffected to all matching widget messages for the various languages.

This operation will only include forms for which the international check control of the form properties dialog box, has been activated.

AddForm
Selection of this menu option allows form components to be added to a project model in conjunction with the project component description dialog box.

The project model name defined in this way may contain wild card characters, in which case a multiple form add operation will result.

AddSource
Selection of this menu option allows new source file components to be added to a project model via the project component description dialog box. Source file components allow both existing and newly created standard source files to be attached to a project model.

The project model name defined in this way may contain wild card characters, in which case a multiple source add operation will result.

AddImage
Selection of this menu option allows image componets to be added to a project model in association with the project component description dialog box.

The project model name defined in this way may contain wild card characters, in which case a multiple image add operation will result.

Executeur
The behaviour of this button depends on the nature of the project component.

For all form components and all abal source files this icon button provides access to the ABAL 3 Runtime and Debug Parameters dialog box allowing execution and debug of ABAL 3 programs from within the ABAL 3 development environment.

For files with either of the .mf or the .mak extensions of named makefile this button will activate the Project Builder dialog box allowing consultation of the project build options prior to launching the update or make operation.

Editor
This icon button provides access to the Editor Configuration dialog box for modification of editor parameters.

Translate
This icon button provides access to the ABAL 3 Translator Parameters dialog box allowing specification of translation parameters prior to the translation operation.

Linker
This icon button gives access to the ABAL 3 Linker Configuration dialog box for the configuration and activation of ABAL module linking.
