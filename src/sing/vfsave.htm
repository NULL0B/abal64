accept fontsave

This visual form will be presented as a result of the selection of the generic menu option File Save whilst the Font Editor is active allowing inspection and modification of the file name to which the current font save operation will apply.

FileName
This edit field allows the name of the font file to be specified.

Accept
By pressing the ACCEPT button the indicated file name will be used as the target for the font save operation.

Select
The Select button provides access to the file selection dialog frame allowing prompted selection of the font filename.

Cancel
By pressing the CANCEL button the font save operation will be abandoned.
