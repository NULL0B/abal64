Font Editor

The font editor of the ABAL 3 development environment gives access to the character font data of the nine text fonts available for use in the design of application forms and document models. The font to be edited may be chosen by clicking on the font tool button in the tool bar on the right of the screen. This will cause the font data to be made accesible to the font editor mechanisms. The font editor window, in the desktop area of the screen, comprises two data inspection frames several edit fields and a collection push buttons. An font specific menu option will also appear on the main menu bar. These interface widgets allow complete control over the various tasks involved in font design and maintanance.

ToolBarControl
The display of the various model editor tools bars is controlled by this button. When the arrow points upwards a click on this button will cause the display of the font selection tool bar to be inhibited. The arrow button will then point downwards in which case a click will cause the font selection tool bar to be redisplayed.

Character
This edit field displays the real textual representation of the character currently accessible in the font character data frame.

DecimalValue
This edit field displays the decimal value of the character in the font data frame.

HexValue
This edit field displays the hexadecimal value of the current character .

CharacterWidth
This edit field displays the actual pixel width of the character currently being edited in the character data frame. For proportional fonts this value will vary from character to character. For fixed fonts all font widths will be the same value.

FontHeight
This edit field displays the overall font character matrix height and is in most cases equal to the individual character matrix height.
By clicking in this edit field keyboard focus will be received by this edit field allowing modification of the font height value.

FontWidth
This edit field displays the overall character width of characters in the font. For fixed fonts individual characters will all be the same width as the font width.
For proportional fonts all character width values will be not greater than this value.
A mouse click in this edit field, will transfer the keyboard focus allowing manual modification of the font width.

CharacterHeight
This edit field displays the height of the character matrix in pixels.

FontBytes
This edit field displays the individual pixels which make up the currently selected character of the font. Individual pixels states may be changed allowing the final form of the character to be modified. This edit frame is the active focus area of the font editor and reacts to keyboard and mouse events as described in the following table.

Leftmoves the pixel cursor leftwards one pixel in a cyclic fashion with line wrap around over the left edge.
Rightmoves the pixel cursor rightwards one pixel in a cyclic fashion with line wrap around over the right edge.
Downmoves the pixel cursor downwards one pixel in a cyclic fashion with column wrap around over the bottom edge.
Upmove upwards one pixel in a cyclic fashion with column wrap around over the top edge.
InsertInserts a blank column of pixels at the current pixel column position pushing all pixel data towards the right edge with destructive overspill of pixels comprising the right edge column.
DeleteDeletes the column of pixels at the current pixel column position dragging all pixels columns, up to the right edge, one column leftwards and inserting a blank column of pixels at the right edge.
Ctrl NInsert a blank row of pixels at the current pixel row position pushing all pixels rows downwards towards the bottom edge with destructive overspill of all pixels comprising the bottom edge row.
Ctrl YDeletes the row of pixels at the current pixel row position dragging all pixels rows below upwards and inserting a blank row of pixels at the bottom edge.
PageDownSaves the current character pixel data to the font memory and loads the next character position from the font table into the pixel data frame for modification.
PageUpSaves the current character pixel data to the font memory and loads the previous character position from the font table into the pixel data frame for modification.

In addition to the above keyboard events, the pixel cursor may be positioned directly using the mouse. A left button click event will activate a pixel in foreground colour whilst the right button will clear the pixel value to the background colour.

NextCharacter
Clicking on this button is equivalent to the page down keyboard operation, saving the pixel matrix to the font table and reloading the next character from the table to the pixel matrix frame for modification.

FontPage
This edit frame displays the current page of characters comprising the current font. Characters are displayed in ascending numerical order up to the end of the ascii table. Clicking on a character in this frame allows its pixel data to be made available in the pixel data frame for modification. The current contents of the pixel data frame will be overwritten and lost if they have been modified since being loaded from font memory into the data matrix.

NextPage
By clicking on this button the Font page display may be moved forward by half a page of characters.

FontName
This edit field displays an identification string of fifteen characters in length that identifies the actual nature of the font. For example Arial 18, Times 24 etc.

PaletteControl
The display of the colour palette bar to the right of the screen is controlled by this button. When the arrow points upwards a click will cause the display of the palette bar to be inhibited. The arrow button will then point downwards in which case a click will cause the palette bar to be redisplayed.

Palette
The Palette bar displays all colours of the currently loaded palette and allows colour selection by a simple click on the required colour. A click with the left button will set the current foreground colour used to represent pixels which contribute to the form of the character in the font data pixel matrix. A click with the right button will set the current background colour used to represent transparent areas of the character data pixel matrix. The effect of changing either colour will take effect when the pixel matrix is reloaded as a result of a page up or page down keyboard event.

File
Selection of this menu options gives access to the generic file operations menu whose options are described here below.

New
This menu option has no effect whilst the font editor is active.

Save
This menu options performs a quick save of the current font memory buffers back to the font file without prompting for user confirmation.

SaveAs
Activation of this menu option gives access to the corresponding font save parameter input dialog box prior to the eventual font save operation.

Open
This menu operation gives access to the corresponding font load parameter input dialog box allowing a new font file to be loaded into the current font memory buffer.

Production
This menu option is an alternative and identical access to the font image file export routines giving access to the font export dialog frame prior to the operation.

Drop
This menu option has no effect whilst the font editor is active.

Quit
This menu option should be selected if you wish to leave the ABAL 3 development tool. If unsaved modifications are detected amongst any of the open project models you will be prompted via the exit check dialog box for the appropriate action required to be taken prior to return to the host operating system or calling process.

View
Selection of this menu option of the main menu gives access to the view or tool operations menu whose options are described below.

Run
Select this option in order to gain access to the debug and runtime control dialog box allowing abal applications to be run, tested and debugged from within the development tool.

Text
Selection of this menu option gives explicite access to the standard text editor for the current model component.

Image
Activation of this menu option gives explicite access to the bitmap editor. allowing inspection and modification of the model image.

FontEditor
Select this menu option to gain access to the font editor.

Properties
Selection of the properties menu option gives access to the 3D component properties dialog box allowing the various properties to be inspected and eventually modified.

Visual
Selection of this menu option gives access to the component specific visual model editor. For forms and document models this will be the visual forms editor. For Image components this will be the bitmap editor and for Source components this will be the standard text editor.

TreeView
Selection of the treeview menu option toggles the treeview mechanism between visible and hidden states.

Application
Selection of this menu option returns control to this project model editor from the vrious other model editor views.

Event
Whilst the font editor is active the popup menu controlled by this option is inhibited.

Options
Selection of this menu options gives access to the general operations menu the options of which are described here below.

Parameters
Selection of this menu option gives access to the General Parameters dialog box allowing the various editor parameters to be inspected and modified. These parameters will be used as default values for all subsequent new widget and other model components as they are created.

Refresh
Selection of this menu option simply causes the development screen to be redraw for any particular reason.

Configure
Activation of this menu option gices access to the Configuration Parameters of the ABAL 3 development tool. These configuration values are read from the configuration file, normally sing.cfg, during the start up and construction phase of the development environment.

ScreenCapture
This menu option may be activated in order to perform a screen capture operation. The entire screen will be captured to a virtual bitmap then the image save dialog box will be presented allowing the target image type and file name to be specified.

Animate
Selection of this menu operation gives access to the Animation Record and Replay control dialog box. Animation recording allows user input events, equally keyboard and mouse events, to the development environment to be recorded to an animation script file. Animation replay allows a recorded script to be re-enacted performing the identical sequence of input events. These features may be used to create input macros or for tutorial purposes.

FontMenu
Selection of the font option of the main menu gives access to the following popup menu of font editor specific functions.

Copy
Selction of this menu option copies pixel data of the character currently in the pixel data frame to clipboard memory.

Paste
Selection of this menu option copies pixel data stored to the font clipboard back into the pixel data frame overwriting the contents of the frame.

Recalculate
Selection of this menu option performs recalculation of minimum column widths for all characters comprising the font resulting in a font of type proportional spacing.

SemiGraphics
Selection of this menu option performs regeneration of the semigraphic characters of the font. These character forms are preprogrammed and will be generated to fint the current font dimensions. This is important if the overall font dimensions have been changed either increasing or decreasing sizes to ensure that semigraphical characters are well formed and consistent.

Clear
Selection of this menu option causes the pixel data matrix to cleared from the current pixel cursor position downwards to the bottom right corner. All pixels above and to the left will remain unchanged.

Fill
Selection of this menu option causes pixels data matrix to be foreground colour filled down from the current pixel cursor position to the bottom right corner of the pixel matrix. All pixels above and to the left will remain unchanged.

Import
Selection of this menu option gives access to the font import dialog box allowing specification of an image filename to be used as the source for the font import operation.

Export
Selection of this menu option gives access to the font export dialog box allowing export of an entire character font to an image file.

Extract
This menu option allows groups of characters to extracted from other font files and copied into the current target font. Selection of this menu option will give access to the Font extraction dialog box allowing definition of the parameters controlling this operation.

Invert
Selection of this menu causes the pixel values of the pixel data matrix to be inverted. Active, foreground pixels becoming background pixels and vica versa.

Paper
Selection of this menu option causes the nature and colour of the background of the font character page to be toggled between plain white and shadowed grey allowing the font characters to be tested in the two different conditions.

Executer
This icon button provides access to the ABAL 3 Runtime and Debug Parameters dialog box allowing execution and debug of ABAL 3 programs from within the ABAL 3 development environment.

Editor
This icon button provides access to the Editor Configuration dialog box for modification of editor parameters.

Translate
This icon button provides access to the ABAL 3 Translator Parameters dialog box allowing specification of translation parameters prior to the translation operation.

Linker
This icon button gives access to the ABAL 3 Linker Configuration dialog box for the configuration and activation of ABAL module linking.
