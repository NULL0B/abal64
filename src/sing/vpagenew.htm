This dialog box presents the collection of document parameters required for the production of graphical printed documents. These parameters are save to the document model file along with the form parameters and the collection of document widgets.

TopMargin
As for the left margin this edit field allows definition of the top margin and will be used in conjunction with the following select control providing the units.
LeftMargin
This edit field, in conjunction with the following select control, allows precise control over the dimensions of the left margin allowing exact page positioning of the printed document.
LeftMarginUnits
This select control provides the units of the preceeding margin value.
TopMarginUnits
This select control determines the units of the preceeding top margin parameter value.
DeviceLo
This edit field provides the name of the printer on which the document will be printed. This may be a constant expression or a variable providing the name dynamicly at runtime. Printer names are not platform portable so care should in this respect.
PaperType
This select control allows selection of the paper type from the following table

A4
A3
A2
A1

The page size, in conjunction with the print density, will determine the number of bits used to represent one pixel and consequently will influence the colour or greyscale depth.

PrintResolution
This select control allows selection of the print density from the following list

75 dpi
100 dpi
150 dpi
300 dpi
600 dpi
1200 dpi

PageLayout
This select control allows definition of the page layout, portrait or vertical, landscape or horizontal.
FeedOpt
This select control allows selection of the way in which page feed characters are to be used. They may be generated prior to printing the page, upon completion of the page or simply inhibited.
Preview
When this check control has been activated then page preview print control code will be generated to allows screen preview of the document when the printer name has been set to space.
PageAlign
This select control allows page alignment to be specified. Page alignment will be applied when the overall output is smaller than the target surface and controls the distrobution of the surplus.
Accept
By clicking on this button you will confirm the collection of document parameters.
Cancel
This push button allows abandon of any modifications prior to leaving the document parameters dialog box.
