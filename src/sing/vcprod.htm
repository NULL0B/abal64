accept c production

This visual form will be presented for the inspection and modificat ion of the various parameters required for the generation of C lang uage source when the forms model language flag is set to C or C++.

FileName
The C Target Filename edit field allows the full path and file name of the production target to be specified.
auto53
The IDENTIFIER edit field allows the class name for C++ production or the function name for C generation to be specified.
IsProgram
When the PROGRAM check box has been checked then C program startup code will be generated to the production target. Otherwise, when unchecked, only the forms related functions or class code wil l be generated.
Accept
By pressing the ACCEPT button the C production operation will be performed to the indicated production target.
Cancel
By pressing the CANCEL button the C production operation will be abandoned.
Inclusions
The INCLUSIONS edit field allows the list of files required to be included for the correct compilation of the production target.
Edit
By pressing the EDIT button the production target file will be made available for inspection via the Standard Abal Text Editor mecha nism.
Select
The Select button allows access to the file selection dialog frame allowing prompted selection of the filename.
