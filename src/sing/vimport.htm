accept font import

This visual form will be presented for the confirmation or modifica tion of the font dimensions during the font import operation.

CellWidth
The CELL WIDTH edit field provides the value calculated by dividing the overall image width by 16, the number of character cells wide the image is assumed to be. This may not be an appropriate width to be used as the default base character width and may need to b e modified.
CellHeight
The CELL Height edit field presents the character cell height as ca lculated by the overall image height divided by 16, the number of r ows of characters assumed to constitute the image. This value may n eed to bemodified to provide a realistic base character cell height .
Auto4868
If neither the CELL WIDTH nor CELL HEIGHT fields have been modified and their default values correspond to the dimensions of the characters then the AUTOMATIC button may be pressed to activa te the auto importation of the entire font from the image.
Prompted
If either of the font dimension edit fields have been modified it i s recomended to press the ASSISTED button to be presented with the individual matrix data items in order to be able to decide as to whichfont entry the correspond, by pressing the appropriate cha racter key on the keyboard.
Cancel
If the CANCEL button is pressed the font import operation will be abandoned.
