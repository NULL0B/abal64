file selector

This dialog frame provides a centralised file selection mechanism for all dialog frames requiring filenames. An individual path storage zone is associated with each of the file types to be maintained and presented across succesive file selection operations of the same file type.

Result
This edit field displays the current result value of the file selection dialog frame.

Accept
By pressing this button the current value of the result edit field will be returned to the caller of the file selection mechanism.

Find
By pressing on this button the combined path and file type criteria will be combined and displayed in the file seclection value. This resulting search pattern will be used to perform the search operation whose results will be displayed in the main output table.

Delete
By pressing the delete button the file represented by the current result edit field may be deleted.

Cancel
By pressing this button the file selection operation will be abandoned.

FileType
This select control provides a list of file types extensions.

Path
This edit field allows a path name to be specified. The path will be used in conjunction with the file type value giving rise to the file search pattern.

FileSelection
This edit filed displays the current search pattern comprising the path field and the file type extension. It may be modified by hand prior to pressing the find button to perform the search operation.

FileNames
This table field displayes the results of the most recent file selection operation. By clicking on a filename it will be copied into the result field.

FileInfos
This table field displayes the results of the most recent file selection operation. By clicking on a filename it will be copied into the result field.
