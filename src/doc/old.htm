ABAL 3 : Object Linker

Command Line Options

-ffilename read command line options from the specified filename.
-zt activate target trace, prints target sector data during program building phase.
-zs activate source trace, prints information during source module loading and resolution phases.
-pprogram or module object type check only.
-vdisplay information during various phases.
-dfilenamespecify target filename.
-rfilenameactivate report production to html filename
-mpreserve debug symbolic information.
