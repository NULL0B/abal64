Abal Product Builder : Version 1.1a.0.01

OptionDescription
-ssourceThis is mandatory and must provide the full path and filename of the complete product package that is to be wrapped by the product builder.
-stargetThis is mandatory and must provide the full path and filename of the product package that is to be unwrapped the installation code produced by the product builder.
-bpreinstallThis is optional and allows preinstallation activity to be defined.
-iinstallThis is mandatory and must provide the full syntax of the installation command and any required options. The target filename will be concatenated to the string separtaed by white space.
-apostinstallThis is optional and allows postinstallation activity to be defined.
product This is mandatory and provides the name of the product package binary auto installable executable program.
