widget reference locator

This dialog frame will be displayed as a result of the activation of the forms editor menu option Widget.Locate and will present the list of forms instances, widgets and their methods from which the selected widget is referenced.

Target
This edit field provides the name of the widget for which the list of references is to be located.

RefList
This control displays the actual list of references to the target widget. These may be from widget methods or from forms methods of any of the forms of the current project. The Widget Properties dialog box, of the corresponding reference may be activated by clicking with the LEFT button of the mouse on the required widget reference expression.

RefBar
The scroll bar may be used to scroll amongst the list of references if this exceeds the visual page.
