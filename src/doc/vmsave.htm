accept modelsave

This visual form will be presented to allow inspection and modification of the various parameters associated with the Application Model Save operation.

FileName
This edit field presents the full path and file name of the file to which the application or project model is to be saved.

Accept
This button, when pressed, confirms the operation and the Application Model and its components will be saved to disk.

Select
This button allows the target model filename to be selected using the standard file selection mechanisms of th visual design tool.

Cancel
By pressing this button the Application Model save operation will be abandoned.

Save Images
If this check box has been checked then out-of-date model images will be saved to their corresponding image file.

Save Forms
If this check box has been selected then out-of-date forms data will be written to the corresponding disk file during the model save operation.

Save All
If this check box has been selected then all form data and image data will be considered to be out-of-date consequently causing them to be rewritten to disk if the corresponding check box controls have been selected.
