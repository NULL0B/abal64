accept visual colour

This dialog frame will be presented as a result of clicking on either the foreground text colour pushbutton or the background colour pushbutton of the Widget Properties dialog box. The dialog box allows selection of the Two colour values from the current palette of colours.

Accept
The currently selected colour values, both for the text and for the background, will be returned to the collaing dialog box by pressing on this button.

fgcolour
This edit field displays the numerical value of the current text or foreground colour. A block of colour to the right of this edit field shows the effective colour represented by this value.

bgcolour
This edit field displays the numerical value of the current background colour. A block of colour to the right of this edit field shows the effective colour represented by this value.

Cancel
If this button is pressed the dialog frame will exit leaving the original colour values, discarding any new values selected.

target
This edit field presents for selection the 256 colours of the current palette. The current value of the foreground or text colour may be selected by clicking with the LEFT button of the mouse on the desired colour of the palette. The current value of the background colour may be selected by clicking with the RIGHT button of the mouse on the desired colour of the palette.
