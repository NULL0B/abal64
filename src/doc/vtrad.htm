FileName
The SOURCE edit field presents the full path and filename of the source file to be translated.

Translator
The TRANSLATOR edit field presents the full path and file name of t he ABAL Translator to be used to translate the source.

Debug
The DEBUG check box, when checked, activates translation with gener ation of a list for an eventual debug operation.

Trace
The TRACE checkbox activates the collection of errors encountered during translation for use by the standard ABAL trace mechanism.

Verbose
When the VERBOSE checkbox is checked the translation operation will display textual information abouts its progression to the releve nt text screen output device or window.

List File
When the VERBOSE checkbox is checked the translation operation will display textual information abouts its progression to the releve nt text screen output device or window.

Optimisation
The OPTIMISATION check box activates the translator optimisation mechanisms.

Compact
The OPTIMISATION check box activates the translator optimisation mechanisms.

Accept
By pressing the ACCEPT button the translation operation will be act ivated with the indicated parameters.

Cancel
The CANCEL button allows the translation operation to be abandoned.

Inclusion
The INCLUDE PATHS edit field presents the various include paths to be used for the resolution of the various include files required for the translation of the source. The command line switch of -I or /I is not to be included as it will be added by the transl ation mechanism prior to the activation of the translator.

Edit
By pressingthe EDIT button the SOURCE file name will be made available for inspection by the Standard Text Editor Mechanisms

Select
The Select button allows access to the file selection dialog frame allowing prompted selection of the filename.
