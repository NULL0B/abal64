accept variable inspection

This dialog box will be displayed in response to a variable inspection request whilst the hexadecimal switch is set to false.

Variable
This edit field shows the variable identification for which this inspection mechanism has been activated.

FirstDim
This edit field allows a value to be specified to be used as the primary index for dimensioned variables.

SecondDim
This edit field allows a value to be specified to be used as the secondary index for dimensioned variables.

Liste
This data table displays the various different variables and their values corresponding to the same variable name. This situation may arise as a result of object source translation or as a result of modular linking of objects with identical named global member variables. Any of the values may besent for memory dump inspection by clicking with the LEFT button of the mouse on the corresponding entry in this table.

Slider
Where the list of variable values exceeds the page size, the scroll ba may be used to page through to the subsequent values.
