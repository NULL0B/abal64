accept form syntax

This dialog box allows the composition of widget method text expressions using class, widget and method selection lists. The widget list will be drawn from the current active visual forms model in conjunction with the widget class selection filter. The memeber list will be built to match the selected widget class. Resulting expressions when accepted will be directly injected into the current editor text method source or document at the current text cursor position.

Result
This edit field collects the result of the syntax composition operation. It may be modified by hand or by clicking on the compose button. The value of this field will be returned and injected into the editor textfile at the current cursor position when the accept button is pressed.

ClassList
This select control provides the list of widget classes allowing selection of a widget type. The widget type will be used to prepare the list of all matching widgets.

WidgetList
This select control provides the list of widgets matching the current widget class selection.

MethodList
This select control provides the list of methods or atributs available for the current widget selection. The member list contents is widget class dependant.

Concatentate
When this check control is active the compose button will concatenate the new syntax expression with the current result contents. This switch is disabled by default on entry to the syntax composer but will be activated by the first composition operation as a result of clicking the compose button. If the expression is to be reset and rebuilt reset this check control prior to pressing the compose button.

Accept
By clicking on this button the current contents of the result field will be accepted and returned as the result of the operation.

Compose
By clicking on this button the current selections will be applied to compose the result field. The concatenation control will be activated on completion of this operation.

Cancel
By clicking on this button the widget syntax composition operation will be abandoned.
