accept fontload

This visual form will be activated in response to the generic menu option File Open being selected whilst the Font Editor is active and allows the name of a font file to be specified or selected prior to being loaded into the corrsponding font slot ofthe font editor and font management mechanisms.

FileName
This edit field allows the font filename to be specified.

IsAnsi
This control switch allows the font loading ansi conversion mechanisms to be disactivated when set to white on red background otherwise, blue on white, the standard ANSI ISO8859-1 to PROLOGUE character set conversions will apply. This may be activated to allow loading of international font data without interpretation. Only the first 256 characters will be loaded.

Accept
By pressing the ACCEPT button the indicated font file will be loaded into the current font slot.

Cancel
By pressing the CANCEL button the font load operation will be abandoned.

Select File
The Select button provides access to the file selection dialog frame allowing prompted selection of the filename of the font to be loaded.

System Font
This button provides access to the system font selection dialog frame allowing prompted construction of system fontname syntax for the selection of system fonts to be loaded.
