Criteria

Replace
This edit field allows the search pattern to be specified.

Ignore Case
When this control is active the search and replace operation will ignore the case of the search expression, otherwise the exact case of the search expression will be respected.

Reverse Search
When this control is active the search and replace operations will occur towards the start of the file otherwise it will proceed towards the end of the file.

Replace All
When this control is active the search and replace operation will be performed in the required direction for all occurance encountered up to the end or start of the file.

Replace
The editior search and replace operation will be launched or repeated by clicking on this button.

Quit
This button allows the current search and replace operation to be abandoned.
