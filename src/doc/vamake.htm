ToolName
This edit field provides the full path and filename of the ABAL makefile build utility amake.

FileName
This edit field provides the full path and file name of the project description file or makefile.

Output
This field allows an output or report file name to be specified. If this field provides a value it will be used to collect al messages output by the project update operation and will be presented for inspection upon completion.

Define
This edit field allows project build options to be specified. These are usually variable or definition names which will be used to control the update operation.

Force Build
When this check control is active the update operation will be an unconditional rebuild of all targets. Otherwise target and source dates will be compared and only out of date targets will be rebuilt.

Show Dependance Dates
When this check control is active target and source file dates and times will be generated to the output log if this has been requested.

Touch Only
When this check control is active out of date targets will be date and time stamped instaed of being rebuilt. This may be used in conjunction with the force build flag causing all target date and time stamps to be refreshed.

Accept
This project build or update operation will be activated by clicking on this button.

Edit
By pressing this button the makefile named will be made available for inspection by the Standard Text Editor Mechanisms

Cancel
If this button is pressed then the update or build will be canceled.
