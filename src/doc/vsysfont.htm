system font selector

  1. Selection Criteria
    This page presents the complete collection of fields required for the selection of X/11 system font files.

    • foundry
      This edit field allows the font foundry or provider name to be specified. If left blank it will default to star allowing selection of any foundry name.

    • slant
      This control allows the font slant to be chosen.

    • charset
      This edit field allows the character set name to be specified. This will default to any when left blank.

    • setwidth
      The font width characteristics may be selected by this control.

    • average
      You may specify here a required average character with. Again zero values have a special meaning.

    • spacing
      This control allows selection of the font spacing type between fixed and proportional.

    • vresdpi
      This control allows selection of the vertical resolution for which the font originally was intended.

    • hresdpi
      This control allows selection of the horizontal resolution for which the font originally was intended.

    • points
      This edit field allows the required dimensions of the font to be specified in points or rather tenths of a point. This field will default to any but be careful of zero values which have a special meaning.

    • pixels
      This edit field allows the required pixel dimension of the font to be specified. This field will default to any but be careful of zero values which have a special meaning.

    • weight
      This control allows the selection of the required font weight.

    • family
      This selection control offser the complete list of font family types currently available.

    • Syntax
      This edit field displays the resulting selection syntax comprising the various fields as will be presented to the font selection manager to provide the selection result.

    • resultname
      This edit field shows the currently selected font name that may be returned to the calling process by clicking on the validation button to the right.

  2. Selection Result
    This page presents the list of system fonts matching the constructed font selection syntax.
