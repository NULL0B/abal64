Text Editor

The internal text editor of the ABAL 3 development environment is an greatly enhanced version of the standard ABAL text editor known as AED.
It has been reworked to allow tight integration into the new development environment and to offer the following series of new features :

  1. Mouse driven operation
  2. Vertical scroll bar positioning
  3. Syntaxical colouration of text content with an ABAL, C and HTML orientation.
  4. Powerful Search mechanisms.

The behaviour of the editor is identical to the original ABAL editor as far as the video mode is concerned. The escape key however opens the command window in which standard AED command sequences are possible. The escape key in the command window performs closure of the window and connot therfore be used for the concatenation of command sequences. To this effect, since the standard editor replace command required the escape key to be used to separate the search and replace strings, menu driven search and replace versions of these commands are available instead.

The standard insert,indent and basket messages have been replaced by the new switch controls in the main menu bar.

Information concerning the configuration of the editor may be found in the section relating to the menu option Options.Configure in this document or that of the other model editors.

The internal text editor will be activated to fulfill the various needs for text editing as follows :

  1. Activation of the menu option View.Text for forms or source file project components.
  2. Activation of the menu option Event.ItemDocument from the forms editor.
  3. Activation of the menu option Event.FormDocument from the forms editor or the project modelling tool.
  4. Activation of an Edit button in one of the secondary dialog boxes.
  5. By selection of an event method from the main menu from within the forms editor.
  6. By selection of an event method page from within the widget properties dialog box.
  7. By selection of either the inclusion or modules page from within the abal production dialog box.
  8. When error tracing has been activated for abal translation in the abal translator dialog box, if the translator returns a translation error then the trace file will be read and loaded by the integrated error trace mechanisms. Each error will be prompted for correction using the trace version of this internal text editor, in which case the suplementary trace menu will be main available.

File
Selection of this menu options gives access to the generic file operations menu whose options are described here below.

New
This menu option produces no effect whilst the text editor is active.

Save
This menu options performs a quick save of the current editor text file to disk without prompting the user for confirmation.

Save As
Activation of this menu option produces the same effect as the quick save operation.

Open
When the cursor of the text editor is positioned over a valid filename string in the text document, activation of this menu option will cause the current file to saved and the new file to be opened and loaded into the text editor.

Production
This menu option produces no effect whilst the text editor is active.

Drop
This menu option produces no effect whilst the text editor is active.

Quit
Selection of this menu option allows exit from the text editor and return to the parent context or model. Any modifications that are unsave will be lost unless the return is to the widget properties dialog box or to the master trace file in error tracing mode.

View
Selection of this menu option of the main menu gives access to the view or tool operations menu whose options are described below.

Run
Select this option in order to gain access to the debug and runtime control dialog box allowing abal applications to be run, tested and debugged from within the development tool.

Text
Selection of this menu option gives explicite access to the standard text editor for the current model component.

Image
Activation of this menu option gives explicite access to the bitmap editor. allowing inspection and modification of the model image.

Font
Select this menu option to gain access to the font editor.

Properties
Selection of the properties menu option gives access to the 3D component properties dialog box allowing the various properties to be inspected and eventually modified.

Visual
Selection of this menu option gives access to the component specific visual model editor. For forms and document models this will be the visual forms editor. For Image components this will be the bitmap editor and for Source components this will be the standard text editor.

TreeView
Selection of the treeview menu option toggles the treeview mechanism between visible and hidden states.

Application
Selection of this menu option returns control to this project model editor from the vrious other model editor views.

Event
This menu option is inhibited whilst the text editor is active.

Options
Selection of this menu options gives access to the general operations menu the options of which are described here below.

Parameters
Selection of this menu option gives access to the General Parameters dialog box allowing the various editor parameters to be inspected and modified. These parameters will be used as default values for all subsequent new widget and other model components as they are created.

Refresh
Selection of this menu option simply causes the development screen to be redraw for any particular reason.

Configure
Activation of this menu option gices access to the Configuration Parameters of the ABAL 3 development tool. These configuration values are read from the configuration file, normally sing.cfg, during the start up and construction phase of the development environment.

Screen Capture
This menu option may be activated in order to perform a screen capture operation. The entire screen will be captured to a virtual bitmap then the image save dialog box will be presented allowing the target image type and file name to be specified.

Animate
Selection of this menu operation gives access to the Animation Record and Replay control dialog box. Animation recording allows user input events, equally keyboard and mouse events, to the development environment to be recorded to an animation script file. Animation replay allows a recorded script to be re-enacted performing the identical sequence of input events. These features may be used to create input macros or for tutorial purposes.

Editor
Selection of the project menu options gives access to the following project model specific menu functions

Find
Selction of this menu option gives access to the Text Find dialog box allowing control of the corresponding text editor mechanim.

Replace
Selection of this menu option gives access to the Text Replace dialog fbox allowing control of the corresponding editor text search and replace mechanism.

Syntax
Selection of this menu option gives access to the method editor syntax composer dialog box offering assistance for widget member expression construction via selection lists. Expressions constructed in this way are more reliable.

Search
Selection of this menu option will launch the AED Ctrl Z or external text search operation for the token situated under the editor cursor position using the external abal grep utility AGR. In order to be able to search in files other than those in the current working directory it may be necessary to issue the new editor command Z allowing the search directory and pattern to be defined.

Trace
This option of the main menu will only be available whhen the text editor has been activated in error trace mode as a result of errors being detected upon completion of an abal translation operation. Selection of the menu option will give access to the following popup menu of trace functions.

InsertSwitch
This switch control displays the current state of the editor insert toggle. This is active by default causing all input characters to be inserted pushing existing characters to the right. The editor INSERT key allows this to be toggled between INSERT and OVERWRITE modes.

SelectSwitch
This switch control display the status of the editor text selection mode. The control will light when the editor selection key Ctrl W is pressed, or when the right button of the mouse is pressed, indicating that the text selection mode is active. The switch will go off when the selection mode is disactivated by the Q key which cancels text selection mode returning to standard video mode. For more information concerning this please consult the Abal Text Editor : AED section of the documentation.

LineNumber
This edit field displays the current cursor line number.

Executeur
This button allows the ABAL Debug Utility to be launched for the inspection and correction of ABAL program components.

By clicking with the right button of the mouse the debug parameters dialog box will be presented, otherwise with the left button the debug utility will be direclty launched.

First
Selection of this menu option causes the trace editor to reposition the text editor to the first item in the internal list of trace items.

Previous
This menu options is identical to the standard trace editor Ctrl V keyboard action which causes the trace editor to position the text editor to the previous item in the internal list of trace items.

Widget
Selection of this menu event requests that the widget properties dialog frame of the parent widget of the source production that raised the error condition, be located and presented for inspection and eventual modification. Any modifications performed in this way will be saved direct to disk as a result of clicking on the accept button under these conditions.

Method
Selection of this menu event gives direct access to the widget method text that is detected to be responsible for the production of the source that gave rise to the error condition.

Next
This option, like the standard keyboard Ctrl T action will request that the trace editor position the text editor to the next item from the list of trace items.

Last
Selection of this menu option requests the trace editor to position the text editor to the last item in the internal list of trace items.

Executer
This icon button provides access to the ABAL 3 Runtime and Debug Parameters dialog box allowing execution and debug of ABAL 3 programs from within the ABAL 3 development environment.

EditConf
This icon button provides access to the Editor Configuration dialog box for modification of editor parameters.

Translate
This icon button provides access to the ABAL 3 Translator Parameters dialog box allowing specification of translation parameters prior to the translation operation.

Linker
This icon button gives access to the ABAL 3 Linker Configuration dialog box for the configuration and activation of ABAL module linking.
