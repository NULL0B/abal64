accept linker parameters

This dialog frame will be activated is response to one of the following events :

  • After the translation phase of ABAL model production for which auto translation and linking has not been selected.
  • By clicking on the corresponding tool button of the menu bar of any one of the various model editors.

allowing consultation and modification of parameters which control the ABAL 3 Link phase of the ABAL Production Cycle for production packages which declare modules for linkage.

Precise information pertaining to the ABAL 3 Object Linker may be found in the corresponding document.

    • Linker
      This edit field provides the full path and filename of the abal module linker to be used for the secondary phase of abal program production for visual models which declare subordinate modules.

    • Target File
      This check field, when active, causes the specific target file name to be used rather than the name of the program object amongst the collection of modules.

    • TargetName
      This edit field provides the name of the executable target program file that is to be generated as a result of the link operation.

    • Html Report
      This check box, when checked, activates the generation of the link phase report to the corresponding html file.

    • ReportName
      This edit field provides the name of the file to which the html report is to generated describing the result of the link and providing an exhastive list of components combined to form the target.

    • Debug
      This control may be checked to indicate that symbolic information is to be generated for debug purposes.

    • Trace Input
      When this check control is active detailed trace information, describing the various input modules, will be printed to the text screen.

    • Trace Output
      When this check is active detailed trace information concerning the production of target blocks will be printer to the text screen.

    • Verbose
      When this check control is active information messages will be printed to the text screen during the link operation.

    • Modules
      This edit field displays the list of modules to be linked to produce the abal executable program.

    • IncludePath
      This edit field displays the list of modules to be linked to produce the abal executable program.

    • Accept
      Pres on this push button to accept the parameters and activate the link operation.

    • Cancel
      This button allows the link operation to be abandoned.
