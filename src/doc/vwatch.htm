Sing Version 1.0a

accept watch inspection

This dialog box will be displayed allowing the inspection of memory contents in response to one of the following events :

  1. By requesting inspection of a variable value whilst the hexadecimal switch value is true.

  2. By requesting hexadecimal display of a variable by clicking on its name in either of the variable inspection lists.

  3. As a result of a variable memory watch event occuring during program execution.

Adresse
This edit field displays the hexadecimal linear base address of the memory that may currently be inspected by the hexa dump viewing mechanism. This value may be changed by specifying a valid linear base address.

VariableName
This edit field displays the identification of the variable for which the memory inspection mechanism has been activated.

DumpZone
This table control shows :
  • In the left column the paragraph address
  • In the center column the hexadecimal byte values of the paragraph
  • In the right column, for values greater than space, the character represention of the byte value.

DragBar
The scroll bar allows paging through the memory beyond the linear base adresse. Illegal memory adresses will be silenty ignored and will result in their hexadecimal display defaulting to space.
