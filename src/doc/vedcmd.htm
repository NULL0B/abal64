accept editor command

This dialog box is presented as a result of pressing the escape key from the standard text editor. It represents the command mode of the standard abal editor and allows a command syntax to be entered and executed.

EditorCommand
This edit field allows a standard abal editor command syntax to be entered. For futher information please refer to the standard abal editor documentation.

Accept
By clicking on this button the command specified in the edit field will be performed prior to exit from the dialog frame.

Cancel
This button may be pressed to leave the editor command dialog box.
