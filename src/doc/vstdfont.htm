This push button allows the character following the current to be loading into the character data matrix for inspection and modification.

Copy
The current character matrix will be copied for pasting by clicking on this button. The character dimensions, width and height, are saved with the character data.

Paste
Character matrix data, both pixel values and dimensison, saved by the Copy function will be pasted into the current character cell by pressing this button.

Save
This push button gives access to the Font Save dialog box allowing a target font filename to be specified and accepted. The currently selected font will then be written, in standard portable FMF format, to the file of the indicated name.

ReSize
This push button allows font dimensions to be recalculated or redistributed depending on the sate of the FixedFont selection value.

When FixedFont indicates a fixed pitch font, the global font dimensions indicated by the FontWidth and FontHeight fields will be applied to all character cells, either expanding the characters or clipping them to adapt.

When FixedFont indicates a proportional spaced font, all font character cells will be recalculated to their exact data useage size plus one.

Export
This push button gives access to the < href="visave.htm">Image Save dialog box allowing an image name to be specified and accepted. An image name provided in this way will be used as the name of the image file to which an image of the entire font character set is to be written and saved. The font image will be organised as a standard 256 character set table.

Frame
When this button is pressed the semi-graphical character of the current font will be regenerated respecting the global font dimensions.

Load
This push button gives access to the Font Load dialog box allowing a font filename to be specified and accepted. The currently selected font will then be loading into the current entry of the font table and will become the current font.

Character
This edit field displays the character currently loaded into the character matrix editor on the left.

CharacterHex
This edit field displays the hexadecimal value of the character currently loaded into the character matrix editor on the left.

CharacterDec
The initial value of this edit field is passed as a parameter to the dialog box and displays the decimal value of the character currently loaded into the character matrix editor on the left. This edit field may be modified by hand or by pressing on either of the character cell forward or back buttons below. The final value of this field will be returned to the caller on completion.

FontName
The initial value of this field will be received as a parameter to the dialog box and will be returned to the caller on exit. The selection mechanism allows the font table entry, on which the editor is to work, to be selected. Each of the 256 font table entries may hold be loaded with an individual font.

FontCellSwitch
This switch controls the behaviour of the row insert/delete and the column insert delete functions of the font data editor. When active the operation will be performed for all 256 characters cells of the font. When disabled the function will be performed on the current character only.

FontWidth
This edit field displays the width in pixels of the currently selected font table entry. It may be increased by clicking on the + button to its right or decreased by clicking on the - to its left.

FontHeight
This edit field displays the height in pixels of the currently selected font table entry. The value displayed here may be increased by clicking on the + button to its right or decreased by clicking on the - to its left.

FixedFont
This selection mechanism allows the nature of the font to be altered between fixed and proportional. This flag will be used by the ReSize function.

CrossControl
This switch item controls display of the font matrix editor switching the cursor on and off allowing font character data to be modified using the standard editor keys

KeyDescription
UP ARROWmove up one matrix row, or wrap arround to bottom row.
DOWN ARROWmove down one matrix row, or wrap arround to top row.
LEFT ARROWmove left one matrix column, or wrap arround to right edge.
RIGHT ARROWmove right one matrix column, or wrap arround to left edge.
CTRL YDelete matrix row at the cursor position pulling up rows beneath.
CTRL NInsert matrix row at the cursor position pushing down the current row and all rows beneath.
DELETEDelete entire matrix column at the cursor position pulling left all columns to the right.
INSERTInsert a new matrix column at the cursor position pushing all matrix columns to the right.
SPACEToggle the matrix pixel value under the cursor position between.

Column
This edit field displays the current character matrix cursor column position.

Row
This edit field displays the current character matrix cursor row position.

CharacterWidth
This edit field displays the width in pixels of the current character in the font character matrix. It may be increased by clicking on the + button to its right or decreased by clicking on the - to its left.

CharacterHeight
This edit field displays the height in pixels of the current character in the font character matrix. It may be increased by clicking on the + button to its right or decreased by clicking on the - to its left.
