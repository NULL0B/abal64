accept general options

This dialog box is available from the main menu choice Options and Parameters and allows the various forms and widget default values to be defined for use during widget creation.

Auto save
This check control when active causes model data to be saved to disk after sucessful production operations.

ProductionLanguage
This select control allows selection of the current default value to be used as the production language for new forms models.

ImageFormat
This select control allows selection of the default image production type.

TextFont
This select control allows selection of the current value to be used as the default font number for the creation of new forms widgets.

Alignment
This select control allows the default alignment value to be defined to be used during the creation of new forms widgets.

Bold
This check control provides the default value of the bold atribut or primary option to be used for the creation of new forms widgets.

Underline
This check control provides the value of the underline option to be used in thecreation of new forms widgets.

Shadow
This check control provides the value of the underline option to be used in the creation of new forms widgets.

Grid On
This check control when set activates the forms editor grid lock mechanisms causing all widget modifications it terms of both position and dimensions to be grid locked or snapped into a predefined matrix of positions and sizes.

GridWidth
This edit field provides the x or width factor used by the active forms editor grid lock mechanism.

GridHeight
This edit field provides the y or height factor to be used by the active grid lock mechanism of the forms editor.

TextFont
This push button may be pressed to commit the current default font dimensions to the corresponding grid lock factors width and height.

ThickNess
This edit field provides the default line thickness to be used for the creation of new line widgets.

Lock On
This check control provides the default value of the widget lock to be used in the creation of new forms widgets.

DropTool
When this check control is active, forms widget tools will be dropped, returning to the standard pointer tool, after each new widget item is added to the form. Otherwise the widget tool will remain active for re-use.
