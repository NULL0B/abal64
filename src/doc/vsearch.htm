accept search

This dialog frame will be preseneted in response to the Find option of the text editor Edit menu. It allows the various options of the editor search function to be specified.

Criteria
This edit field allows the search pattern to be specified.

Ignore Case
When this control is active the search will not be case sensitive, other wise the exact case of the search expression will be respected.

Reverse Search
When this control is active the search will occur towards the start of the file otherwise it will proceed towards the end of the file.

Find
The editior search operation will be launched or repeated by clicking on this button.

Quit
This button allows the current search operation to be abandoned.
