accept modelfilename

This visual form will be presented for the confirmation or modification of the application model file name to be loaded.

FileName
Theis edit field presents the full path and file name of the application model to be loaded.

Accept
By pressing this button the operation will be confirmed and the application model will be loaded if it exists.

Select
This push button allows access to the standard file selection mechanisms for prompted selection of the file to be loaded.

Cancel
The application model load operation may be abandoned by pressing this button.
