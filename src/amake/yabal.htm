ABAL++ Wabal Project Convertor

Command Line Options

-v

activate messages

-d

default source path name

This option may be necessary for projects which have not specified the actual master source path.

-e

echo source stream to stdio

-l

list structure report to filename

This option produces an amalgamated list of information from all input projects .

-s

substitute b for a in filenames

This option is useful when filenames in the project file have be prefixed by an MSDOS drive which requires removal or substitution on a global basis.

-r

redirect filename a with path b

This option indicates that all instances of filename a are to be prefixed by the pathname b.

-i

include tool definitions from file

If this option is not specified then default tool definitions are generated by YABAL into the definitions file.
