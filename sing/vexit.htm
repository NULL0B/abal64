accept exit

This visual form will be displayed to notify the potential loss of modifications to the current project.

JustQuit
By clicking on this button the project model and all unsaved modifications will be abandoned and consequently lost.

AbandonExit
By clicking on this button the current operation will be abandoned and control will be returned to the model editor preserving any unsaved modifications.

SaveOnExit
By pressing on this button the unsaved modifications will be save to disk and then the project model will be released.
