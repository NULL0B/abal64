accept form save

This visual frame is presented to allow the filename of a visual form model to be specified or modified priot to being saved to disk.

FileName
This edit field allows the full path and filename of the file to be specified.
Accept
By pressing this button the file save operation will be validated and consequently the model will be saved to the specified file.
Cancel
By pressing this button the model save operation may be cancelled. .
Select
The Select button provides access to the file selection dialog frame allowing prompted selection of the filename.
