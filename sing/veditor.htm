visual text editor

This visual form is a temporary text editor that will be eventually replaced by a more sophisticated mechanism.

Accept
By pressing the ACCEPT button the text of the main edit field will be returned to the called to be stored as the value of the corre sponding entity.
Cancel
By pressing the CANCEL button any modifications performed will be cancelled and the previous value of the calling entity will be preserved.
Body
The main edit field provides a basic text entry function supporting insert and overwrite modes. The standard curser keys perform the ir usual functions as do HOME and END. The key sequence Ctrl A allows the entire zone to be reset .
