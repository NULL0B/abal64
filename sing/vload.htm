accept form load

This visual form will be presented to allow the name of a form to be specified prior to a form load operation.

FileName
This edit field allows the data entry of the full path and filename of the file from which the model is to be loaded.
Accept
By pressing this button the model identified by the filename field will be loaded from disk.
Select
The Select button gives access to the file selection dialog frame allowing prompted selection of the filename.
Cancel
By pressing on this button the model load operation will be abandoned.
