accept cube properties

This visual form will be presented to allow the properites of an application model component to be inspected and modified.

Identity
This edit field allows the generic name of the model entity to be specified.

Depending on the class or nature of the component an extension will be added to the name provided here

FORMThe name token will be postfixed by .gif or .bmp to form the image filename, and by .sng to form the relevant component model filename.
IMAGENo extension will be added since the exact filename is required for image model loading.
SOURCEAgain no extension will be added as the exact filename is required.

Position
The POSITION X, Y, and Z edit fields allow the position within the application model 3d space of the top left hand corner to be inspected or modified.

Dimensions
The VECTOR X, Y and Z edit fields provide the dimensions of the cube along the relevant AXIS in a positive direction from the top left corner of the component cube.

Accept
By pressing the ACCEPT button the various parameters will take effect.

Select
This push button gives access to file selector dialog box allowing prompted selection of the application component.

Cancel
By pressing the CANCEL button any modifications to these parameters will be discarded.

Class
The CLASS edit field provides the name of the application entity class, either FORM, IMAGE or SOURCE.

AutoSize
if the AUTOSIZE check box is checked the VECTOR information will be adjusted to the real size of the image used to represent the application model component. When it is unchecked values provided manually will be respected.

InhibitGif
This check box control may be checked to inhibit the creation of GIF images. In this case they will be save in BMP format instead.
