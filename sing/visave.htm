accept imagesave

This visual form will be presented to allow the full path and file name of the file to which the current image save operation will apply.

Arrival here will be in response to one of the following

Options Screen Capture for the name of the file to receive the captured screen shot image.
Fonts Export for the name of the file to receive the complete font character set image.
File Save As When the BitMap or Image Editor is active for the name of the file to be saved.

FileName
This edit field allows the file name, of the image to be saved , to be specified.

PathSelection
This select control presents the list of image paths currently defined by the configuration variable to this effect.

The value selected here will be over-ridden by any explicite path information provided in the FileName edit field.

ImageFormat
This select control allows the image format to be specified. Currently the only formats that may be produced are BMP and GIF. The later producing mush smaller files but may require longer image save times.

IsTransparent
If this check control is active then GIF images will have their transparence atribut set to the visual design tool standard value ZERO allowing the subsequent image file to be loaded and used for spriting.

N.B.BMP files are unable to make use of this transparence value.

Accept
By pressing the ACCEPT button the indicated filename and path selection will be used as the target for the image save operation of the indicated format.

Select
The Select button allows access to the file selection dialog frame allowing for prompted selection of the image filename.

Cancel
By clicking the CANCEL button the image save operation will be abandoned.
