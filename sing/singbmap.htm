Image Editor

The Image, or bitmap, editor of the ABAL 3 development environment provides a collection of functions for the design and maintanance of image files to be used in the construction of ABAL 3 application programs.

The ABAL 3 development environment and Application runtime libraries may read images from the following image file formats :

  1. BMP
  2. ICO
  3. GIF
  4. JPEG
  5. PNG
Images are loaded into virtual bitmap buffers, also refered to as pixel storage units. These bitmap buffers may be loaded into the bitmap editor for inspection, modification, enhancement and transformation.

In the current version the ABAL 3 development environment may only write images in the following file formats:

  1. BMP
  2. GIF

ToolBarControl
The display of the various model editor tools bars is controlled by this button. When the arrow points upwards a click will cause the display of the tool bar to be inhibited. The arrow button will then point downwards in which case a click will cause the tool bar to be redisplayed.

ImageTools
The Bitmap or Image Editor allows modification of images at a pixel level. To this purpose the following selection of pixel manipulation tools are offered and can be seen presented on the vertical tool bar to the left of the desktop area.

  1. Filled Rectangle.

    An area defined by its top left corner and bottom right corner may be filled with the foreground colour by using the left button or with the background colour by using the right button of the mouse.

  2. Outlined Rectangle

    An area defined by its top left corner and bottom right corner may be Outline with the foreground colour by using the left button or with the background colour by using the right button of the mouse.

  3. Outset Rectangle

    An outset relief frame defined by its top left and bottom right corner using the standard palette relief colours.By pressing the left button a background fill will be applied. By pressing the right button the background fill will be inhibited with only the frame being drawn.

  4. Inset Rectangle

    An inset relief frame as for the above outset rectangle.

  5. Ridge Rectangle

    A ridged relief frame as for the inset and outset rectangles.

  6. Groove Rectangle

    A grooved relief frame as for the above ridge, inset and outset rectangles.

  7. Pixel Pen

    Allows freehand pixel colouring using the left button foreground colour and right button background colour.

  8. Pixel Sampler

    Sets the current foreground, when left button, or background colour, when right button, by sampling the source pixel colour value.

  9. Line

    Draws a line between the two points defining its extremities.

  10. Area Fill

    Fills all reachable and contiguous pixels with the foreground colour,when left button, or the background colour when right button operation.

  11. Outlined Circle

    Draws a circle outline to fit the rectangle defined by its topleft corner and bottom left corner using the foreground colour, left button, or background colour when right button operation.

  12. Filled Circle

    Draws a filled circle fit the rectangle defined by its topleft corner and bottom left corner using the foreground colour, left button, or background colour when right button operation.

  13. Zone Selection

    Allows definition of a source pixel area or zone for the menu option Image Copy or a target pixel coordinate for the menu option Image Paste

  14. Pointer

    This tool does not modified pixels in any way it may be used for movement of the image by dragging on image pixels as follows :

    • Drag with left button downwards : half page up
    • Drag with left button upwards : half page down
    • Drag with left button leftwards : half page right
    • Drag with left button rightwards: half page left
    • Drag with right button downwards : direct to top edge
    • Drag with right button upwards : direct to bottom edge
    • Drag with right button leftwards : direct to right edge
    • Drag with right button rightwards: direct to left edge

PaletteControl
The display of the colour palette bar to the right of the screen is controlled by this button. When the arrow points upwards a click will cause the display of the palette bar to be inhibited. The arrow button will then point downwards in which case a click will cause the palette bar to be redisplayed.

Palette
The Palette bar displays all colours of the currently loaded palette and allows colour selection by a simple click on the required colour. A click with the left button will set the current foreground colour whilst a click with the right button will set the current background colour. When using the various image tools for the modification of pixel values, clicking with the left button will apply the foreground colour whilst clicking with the right button will apply the background colour.

New
Selection of this menu option causes the contents of the current image data to be reset and filled with the current background colour.

This menu options performs a quick save of the current image to its current image file format without prompting the user for confirmation.

This menu operation gives access to the corresponding image load parameter input dialog box allowing a new image file to opened replacing the current image component.

Production
Selection of this menu option gives access to the image save parameter input dialog box as for the file save menu option.

Drop
This menu option has no effect whilst the image editor is active.

Quit
This menu option should be selected if you wish to leave the ABAL 3 development tool. If unsaved modifications are detected amongst any of the open project models you will be prompted via the exit check dialog box for the appropriate action required to be taken prior to return to the host operating system or calling process.

Options
Selection of this menu options gives access to the general operations menu the options of which are described here below.

Parameters
Selection of this menu option gives access to the General Parameters dialog box allowing the various editor parameters to be inspected and modified. These parameters will be used as default values for all subsequent new widget and other model components as they are created.

Refresh
Selection of this menu option simply causes the development screen to be redraw for any particular reason.

Configure
Activation of this menu option gices access to the Configuration Parameters of the ABAL 3 development tool. These configuration values are read from the configuration file, normally sing.cfg, during the start up and construction phase of the development environment.

ScreenCapture
This menu option may be activated in order to perform a screen capture operation. The entire screen will be captured to a virtual bitmap then the image save dialog box will be presented allowing the target image type and file name to be specified.

Animate
Selection of this menu operation gives access to the Animation Record and Replay control dialog box. Animation recording allows user input events, equally keyboard and mouse events, to the development environment to be recorded to an animation script file. Animation replay allows a recorded script to be re-enacted performing the identical sequence of input events. These features may be used to create input macros or for tutorial purposes.

Image
Selection of the Image option of the main menu gives access to the following image editor specific menu functions

Copy
Selction of this menu option causes the selected pixel area of the current image being edited to be copied into the image editor clipboard memory. The selection of a pixel area is performed as follows :
  1. Select the image editor cut frame tool.
  2. Press the left button of the mouse while the mouse pointer is over the top left pixel of the zone to be selected.
  3. Drag the mouse,maintaining the left button pressed, until the mouse pointer tip is over the bottom left most pixel of the zone to be selected.
  4. Release the left button of the mouse.
As a result of this a green cut frame will be left visible defining the selected pixel area.

Paste
Selection of this menu option allows the contents of the image editor clipboard memory to be pasted into the current image at the image editor pixel cursor position. The image editor clipboard memory is maintained whilst the ABAL 3 development tool is active, consequently image portions may be easily cut from one image and pasted into another. If the pixels pasted from the clipboard memory would exceed the left or bottom edge of the target image the excess will be silently discarded.

Cut
Selection of this menu option causes the all pixels from the current pixel curser position down to the bottom right corner of the image to be copied to the image editor clipboard.

Rotate90
Selection of this menu option causes the current contents of the image editor to be rotated 90 degrees in a clockwise direction. The image dimensions will be altered to reflect the final dimensions of the resulting image with the new width equal to the previous height and the new height taking the previous width value.

Rotate180
Selection of this menu option causes the image editor contents to be rotated a full 180 degrees. The image dimensions will naturally be preserved.

Rotate270
Selection of this menu option causes the current contents of the image editor to be rotated 270 degrees in a clockwise direction, or 90 degrees in an anticlockise direction. The image dimensions will be altered to reflect the final dimensions of the resulting image with the new width equal to the previous height and the new height taking the previous width value.

ChangeColour
Selection of this menu option performs a colour substitution. All pixels matching the current background colour are replaced by that of the current foreground colour.

GreyScale
Selection of this menu option causes the entire image to be reduced to the nearest grey scale conversion. This operation is similar to that performed by the graphical printing mechanisms for non colour printer drivers and consequently allows colour conversion checking of a application document image.

VerticalMirror
Selection of this menu option causes the contents of the image editor to be transformed by reflection across the central vertical axis.

HorizontalMirror
Selection of this menu option causes the contents of the image editor to be transformed by reflection across the central horizontal axis.

Dimensions
Selection of this menu option gives access to image editor image dimensions dialog box allowing the image dimensions to be changed.
