This dialog box will be activated in response to one of the following events :

  • The View Run menu option from one of the following model editors :
    1. Application Project Editor
    2. Visual Forms Editor
    3. Tree View Editor
  • By clicking on the corresponding tool button from any one of the various model editors.
allowing the runtime and debug operational parameters to be specified before activation of the corresponding mode of operation.

FileName
This edit field allows modification or selection of the name of the ABAL 3 program to be run or debugged.

CommandLine
This edit field allows any eventual command line parameters to be specified prior to launching the program.

Select
The Select button provides access to the file selection dialog frame allowing prompted selection of the filename of the ABAL3 program to be run or debugged.

IsDebug
This check control switchs the operational mode between debug and runtime. When checked the debug utility will be lauched otherwise the standard runtime module will be used to run the program.

Debugger
This constant edit field presents the name of the ABAL 3 debug utility as defined by the Options Configure Exadb parameter of the Configuration Options dialog box.

DebugRoot
This edit field allows the debug root path to be specified. This value may be required if the source files of the program to debugged have been translated without explicite path information and are not present in the current working directory.

Runtime
This constant edit field presents the name of the ABAL 3 runtime utility as defined by the Options Configure Exa parameter of the Configuration Options dialog box.

Accept
By clicking on this button the requested operation will be launched.

Cancel
By clicking on this button the operation will be abandoned and control will be returned to the parent editor or work environment.
