accept editor

This visual frame will be presented, if it has not been inhibited b y the corresponding check box, whenever an EDIT button is pressed in a relevent visual frame requiring a source or text for to be presented for inspection or modification.

FileName
The FILENAME edit field presents the name of the file to be edited.

EditorName
The EDITOR NAME edit field presents the full path and filename of the program to be used for the editing of all source and text files. If this string is prefixed by the ! character then the internal standard text editor will be used.

RazColour
This colour selection control allows the standard text colour to be selected for the new syntaxical colouration mechansims for ABAL, C and HTML source files.

QuoteColour
This colour selection control allows the colour to be used to display string litteral values to be selected for use by the new syntaxical colouration mechansims.

KeyWordColour
This colour selection control allows the colour to be used for the display of language keywords to be chosen for the syntaxical colouration mechansims of the standard text editor..

CommentColour
This colour selection control allows selection of the colour to be used for the display of comments.

Accept
When the ACCEPT button is pressed the specified parameters will be returned to the caller for storage. When this frame is called from an Edit button actual source editing will be initiated.

Cancel
if the CANCEL button is pressed any modifications to the various editor parameters will be discarded and any pending editing operation will be abandoned.

FrameInhibit
If this checkbox is checked then this dialog box will no longer be presented for validation prior to editing text or source files. The Editor will be simply activated without confirmation.

TextFont
This select control allows the editor text font to be chosen. Proportional spacing fonts are not recommended as the internal editor is built around aed which has a strong character orientation.
