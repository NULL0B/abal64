This visual frame will be activated when the Menu option : OPTIONS CONFIGURE is selected and presents the various system dependant configuration variable values.

FontPath
The FONTPATH variable defines the full path name of the directory to and from which all fonts will be saved and loaded.
ImagePath
The IMAGEPATH variable defines the full path name of the directory to and from which images will saved and loaded.
Palette
The PALETTE variable defines the full path and filename of the colourmap palette file to be loaded by the application.
BaseFont
The BASEFONT variable defines the default base font used by the development tool for its various hardware needs.
BiosMode
The BIOSMODE variable provides the default graphics mode for the development tool. This may be overridden by system dependant mea ns.
FileName
The FILENAME edit field shows the name of the file from which the configuration information was loaded.
Update
The UPDATE checkbox when selected causes the configuration values t o be written to the configuration file.
Accept
By pressing the ACCEPT button, with the UPDATE option chosen the configuration values will be written to the file.
Cancel
By pressing ther CANCEL button any modifications will be discarded.
