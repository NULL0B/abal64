accept animation parameters

This dialog box is presented to allow access to and control over the visual design tool animation mechanisms.

Animation, in this sense, implies the creation and replay of a series of mouse and keyboard events for the demonstration or repetion of a particular feature of the visual design tool.

This dialog frame when made visible or active will allways disactivate any animation script recording or replay currently underway.

FileName
This edit field allows definition of the animation script file to which the animation will be recorded or from which an animation will be replayed.

Mode
This select control allows selection of the nature of the animation operation, either record events or playback.

Accept
Press on this push button to start the desired animation operation.

Animation event recording or replay will be started by the destruction mechanisms of this dialog frame upon receipt of the signal from this push button.

Cancel
By pressing this button the anaimation operation may be cancelled prior to leaving this dialog frame.
